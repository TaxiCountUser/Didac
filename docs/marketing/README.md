# TaxiCount — Material de marketing

Carpeta con el material de venta y comunicación de TaxiCount. Todo son documentos vivos:
amplíalos con tus cifras, testimonios y ángulos propios.

## Índice

- **[resumen-comercial.md](resumen-comercial.md)** — Resumen comercial completo: pitch,
  público objetivo, dolores, funcionalidades en clave de beneficio, diferenciadores,
  modelo de negocio, claims cortos, y **dos guiones de vídeo** (40 s y 60 s).
- **[landing-copy.md](landing-copy.md)** — Copy completo de la landing, listo para publicar:
  SEO/meta, hero, problema, solución en 3 pasos, funcionalidades, dos perfiles, precios,
  retos/referidos, seguridad, testimonios (placeholders), FAQ, CTA y footer. Incluye una
  sección de **notas para el análisis y el gancho de venta**.

## Cómo usarlo

1. Parte de `landing-copy.md` para montar la web y de `resumen-comercial.md` para el vídeo.
2. Sustituye los placeholders `⟨…⟩` (precios, cifras, nombres, ciudades) por datos reales.
3. Refuerza el hero y el CTA con cifras de credibilidad cuando las tengas (usuarios,
   carreras registradas, valoración media…).
